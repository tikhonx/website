Linux Stuff:
Installing .deb packages: sudo dpkg -i DEB_PACKAGE
Removing .deb packages: sudo apt purge PACKAGE_NAME
Removing Repository path: /etc/apt/sources.list.d/official-package-repositories.list (Use Software Sources)

Github in Terminal
1. sudo apt install git
2. -> Create a repository in github
3. -> get a token to log in
4. git clone https://YOUR_NAME:YOUR_TOKEN@github.com/YOUR_NAME/YOUR_PROJECT
// Go inside the folder
5. git init
// Make desired changes. To upload do this:
git add .
git commit -m 'YOUR_MESSAGE'
git push
